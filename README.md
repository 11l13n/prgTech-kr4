# Генератор паролей (КР4, вариант 17)

Учебное приложение на **React**, реализованное в рамках контрольной работы №4 (вариант 17).  
Приложение генерирует случайные пароли с настраиваемыми параметрами и сохраняет историю сгенерированных значений.

---

## Основные возможности

- Генерация случайных паролей
- Настраиваемые параметры:
  - длина пароля (от 4 до 64 символов)
  - использование строчных букв (a–z)
  - использование заглавных букв (A–Z)
  - использование цифр (0–9)
  - использование специальных символов (!@#$%^&* и др.)
- Отображение текущего пароля в отдельном поле
- Копирование пароля в буфер обмена по кнопке
- Сообщение об ошибке, если не выбран ни один тип символов
- История последних сгенерированных паролей:
  - сохранение до 10 последних паролей
  - время генерации для каждого пароля
  - возможность скопировать пароль из истории
  - кнопка очистки истории
- Сохранение настроек и истории в `localStorage`:
  - при перезагрузке страницы сохраняются длина пароля, выбранные типы символов и история

---

## Используемые технологии

- **React** (Create React App)
- **JavaScript (ES6+)**
- **CSS** (кастомные стили в `App.css`)
- **LocalStorage** для сохранения истории и настроек
